What do these CSS properties do? Color, Background-color, Border-color

  • Color: This property sets the color of the text within an element
  • Background-color: This property sets the background color of an element.
  • Border-color: This property sets the color of an element's border.

How does CSS accept color as a value? Provide examples of colors in these styles:

  • Named Colors: You can use a variety of predefined color name. Exampels include Red, Blue & Green
  • Hexadecimal: Hexadecimal color values start with a # and are followed by a six-digit code that represents the red, green, and blue (RGB) values. Examples include #FF0000 for (red) #0000FF for (blue) & #008000 for (green)
  • RGB: The rgb() function allows you to specify a color by its red, green, and blue components, with each value ranging from 0 to 255. Examples include rgb(255, 0, 0) for (red), rgb(0, 0, 255) for (blue) & rgb(0, 128, 0) for (green)
  • HSL: The hsl() function allows you to specify a color by its hue, saturation, and lightness. Examples include hsl(0, 100%, 50%) for (red), hsl(240, 100%, 50%) for (blue) & hsl(120, 100%, 25%) for (green)
